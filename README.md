# 前后端分离个人健康管理系统 java13

## 项目概述

本项目是一款基于前后端分离架构的个人健康管理系统，旨在为用户提供一个全面、便捷的健康数据管理平台。

## 技术栈

- 后端：Spring Boot + MyBatis-Plus
- 前端：Vue2 + ElementUI + ECharts + Vuex
- 数据库：MySQL、Redis

## 功能模块

### 用户管理

- 注册
- 登录
- 系统可视化统计信息首页

### 系统角色

- 普通用户
- 超级管理员

### 主要功能

1. 用户管理：增删改查用户
2. 角色管理：区分普通用户与超级管理员
3. 运动知识管理
   - 新增
   - 修改
   - 删除
   - 搜索
4. 运动详情管理
   - 新增
   - 修改
   - 删除
   - 搜索
5. 用户身体信息管理
   - 修改
   - 删除
   - 搜索
6. 身体信息上传：上传用户身体各项信息
7. 健康评估：通过ECharts可视化分别统计用户的健康信息
8. 运动知识：搜索和查看各项运动知识
9. 体检健康信息管理
   - 编辑
   - 删除

## 系统角色

- 普通用户：可查看、编辑个人资料，上传身体信息，查看健康评估等。
- 超级管理员：具备普通用户权限外，还可以管理用户信息和运动知识。

## 运行环境

- 后端：JDK 1.8+，支持Spring Boot框架的运行环境
- 前端：支持Vue.js 2.x的浏览器环境
- 数据库：MySQL 5.7+，Redis 3.2+

## 部署步骤

1. 准备后端运行环境，配置数据库。
2. 部署后端服务，确保API接口可用。
3. 编译前端资源，部署到Web服务器。

## 目录结构

```
项目名称/
├── backend/       # 后端代码目录
├── frontend/      # 前端代码目录
└── sql/           # 数据库脚本目录
```

## 核心亮点

- 前后端分离设计，易于维护和扩展。
- 基于成熟的Spring Boot和Vue技术栈，确保系统稳定性。
- 集成ECharts，实现健康数据的可视化展示。
- 提供用户角色管理，保障系统安全。
